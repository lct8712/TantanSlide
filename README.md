# 仿探探的滑动效果

## 实现的效果
  - 堆栈效果
  - 栈顶图片移动时，后面几张图片的联动效果
  - 滑动时图片的倾斜效果
  - 放手回到中心后的反弹效果 
  - 靠近屏幕边缘时自动滑出
